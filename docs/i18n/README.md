# i18n

Translations of documentation. The canonical languages are:
- English: [`../README.md`](../README.md)
- 中文 (简体): [`../README.zh-CN.md`](../README.zh-CN.md)

## Contributing a new translation

1. Copy `../README.md` to `README.<lang>.md`.
2. Translate, keeping the same section structure.
3. Add a badge at the top: `[![Docs: <Lang>](https://img.shields.io/badge/docs-<Lang>-<color>.svg)](./README.<lang>.md)`.
4. Add a back-link at the bottom: `> Read [<Lang> docs (README.md)](./README.md) for the English version.`
5. Open a PR.

We accept any language. Keep section headings semantically equivalent (don't paraphrase the structure).
