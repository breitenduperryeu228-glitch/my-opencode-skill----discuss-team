## 🚀 Quick Start (TL;DR)

```bash
/discuss-team <your-idea-here>
```

That's it. The skill handles config loading, persona activation, debate, and plan generation.

See [`quick-start/invocation.md`](./quick-start/invocation.md) for trigger phrases (slash + natural language).
